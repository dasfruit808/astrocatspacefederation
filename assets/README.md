Place the following files in the `assets` folder:
- player.png: 32x22 pixel art ship (retro style, light blue/white shades)
- alien.png: 24x18 pixel art alien (purple-green retro style)
- alien2.png: Another distinct 24x18 alien sprite (blue/gray style)
- bullet.png: 7x15 pixel art (yellow/white for player)
- abullet.png: 7x15 pixel art (red or green for alien bullets)
- explosion.png: Not used directly, but can be simple pixel burst, otherwise explosions are drawn with code
- shoot.wav: 8-bit shoot sound effect (e.g., from sfxr or open assets)
- explosion.wav: short retro explosion sound effect
- gameover.wav: fail/game over sound

If you need assets, check [Kenney.nl](https://kenney.nl/assets?q=space), [OpenGameArt](https://opengameart.org/art-search?keys=invader), or generate with [jsfxr](https://sfxr.me/). Name them as above. Keep asset file sizes small for instant play.
