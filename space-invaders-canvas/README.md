# Space Invaders Canvas

## Overview
This project is a full-featured side-scrolling Space Invaders shooter game built using HTML5 Canvas and JavaScript. It includes player movement, alien waves, collision detection, explosions, scoring, and sound effects, providing an engaging gaming experience.

## Project Structure
```
space-invaders-canvas
├── index.html        # HTML structure of the game
├── style.css         # Styles for the game
├── script.js         # Main game logic
├── assets
│   ├── sprites       # Placeholder for sprite images
│   │   └── .gitkeep
│   └── sounds        # Placeholder for sound files
│       └── .gitkeep
├── README.md         # Project documentation
├── .gitignore        # Files to ignore in version control
└── LICENSE           # Licensing information
```

## How to Play
1. Open `index.html` in your web browser.
2. Use the arrow keys to move your player ship left and right.
3. Press the spacebar to shoot bullets at the invading aliens.
4. Avoid getting hit by alien projectiles and try to destroy all aliens to score points.

## Setup Instructions
1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd space-invaders-canvas
   ```
3. Open `index.html` in your preferred web browser to start playing.

## Features
- **Player Movement**: Control the player ship using keyboard inputs.
- **Alien Waves**: Aliens move in waves and shoot projectiles at the player.
- **Collision Detection**: Detect collisions between bullets and aliens, as well as between alien projectiles and the player.
- **Explosions**: Visual effects for when aliens are destroyed or when the player is hit.
- **Scoring System**: Keep track of the player's score based on the number of aliens destroyed.
- **Sound Effects**: Audio feedback for shooting, explosions, and game events.
